## What is Knowledge Stack?

Knowledge Stack is an API-first platform that lets you **ingest**, **organize**, **search**, and **converse** over your documents. Upload files, and Knowledge Stack automatically structures them into a searchable knowledge base with built-in AI capabilities — all accessible through a REST API.

### What you can build

- **AI-powered search** — Let your users search across thousands of documents using natural language. Knowledge Stack handles chunking, embedding, and retrieval so you don't have to.
- **Document Q&A** — Build conversational interfaces that answer questions grounded in your documents. Stream AI responses in real-time with full source attribution.
- **Knowledge bases** — Organize documents into a hierarchical folder structure with versioning, sections, and fine-grained access control.
- **Multi-tenant applications** — Each tenant gets isolated data, roles, and permissions out of the box. Build SaaS products without managing authorization yourself.

### Core capabilities

| Capability | What it does |
|-----------|-------------|
| **Document ingestion** | Upload PDFs, DOCX, PPTX, Markdown, and plain text. Files are automatically converted, chunked, and embedded for search. |
| **Hierarchical organization** | A Unix-inspired path system where folders, documents, versions, sections, and chunks form a navigable tree. |
| **Semantic search** | Vector similarity search with metadata filters and authorization-aware scoping — users only see results they have access to. |
| **Conversational threads** | Chat threads with streaming responses, subthreads, and document-anchored conversations. |
| **Roles & permissions** | Three roles (Owner, Admin, User) with path-level permission grants for fine-grained access control. |
| **SSO & OAuth2** | Google SSO, tenant-specific OpenID Connect, and email/password authentication. |
